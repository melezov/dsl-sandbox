History concept is used to log changes made to an object. Using history provides information about object values in each point in time a change was made.

This DSL defines a single root with history. In PHP, we can get a list of all changes made to an object by calling the getHistory method. Each entry in history list consists of:

  • time the change was performed
  • type of performed action (one of INSERT, UPDATE or DELETE)
  • the resulting object value
