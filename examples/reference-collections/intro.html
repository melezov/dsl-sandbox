Property can be defined as a reference collection in the following format: 

type?[]? *name;

If first question mark is included collection can contain null values.

Second question mark defines the property itself as nullable.

Collections in PHP are represented with arrays. When adding new elements to collections, accessing using the [] syntax, (e.g. $object->property[]=...) will result in PHP issuing notice: "Notice: Indirect modification of overloaded property...". Therefore, properties must be assigned in whole, or using the generated helper methods.
