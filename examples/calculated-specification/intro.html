Property can be calculated from specification. Calculated property is a read-only, boolean property. Expression in specification always evaluates to true or false.

Property can be calculated directly from expression (without defining specification) by specifying th expression body after the from keyword in property.

Calculated properties (both from expression and specification) can be referenced from snowflake.

Note that expression in calculated property is evaluated on the server. That means newly created objects must first be persisted in order to access the calculated property value.
